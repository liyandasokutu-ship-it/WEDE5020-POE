## Project Overview
The Youth Empowerment Hub is a fictional non-profit organisation 
established in 2022. This website was built to provide information 
about the organisation's programmes, events, and contact details. 
The website is built using HTML, CSS, and JavaScript.

---

## Website Goals and Objectives
- Provide clear and accessible information about the organisation
- Promote programmes, workshops, and events
- Encourage user engagement and participation
- Make it easy for users to contact the organisation

---

## Key Features and Functionality
- Responsive design for desktop, tablet, and mobile
- Navigation menu linking all 5 pages
- Contact form and enquiry form
- Google Maps embed showing two office locations
- Upcoming events section
- Team member profiles

## Wireframe Description
Due to the wireframe not being included in Part 1 submission,
the following describes the website layout:

- **Navigation Bar** - Logo on left, menu links on right
- **Hero Section** - Large banner with heading and two buttons
- **About Snippet** - Brief intro with read more button
- **Programs Section** - Four cards in a grid layout
- **Events Section** - Vertical list with date boxes
- **Footer** - Three columns with links and contact details
